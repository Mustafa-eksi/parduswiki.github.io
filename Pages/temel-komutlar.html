Yeni başlayanlar için temel Pardus komutları

Paket işlemleri

Depo güncellemek için: 

	sudo apt-get update
Kurulu tüm paketleri güncellemek için güncellemek için: 

	sudo apt-get upgrade
Sistemi güncellemek için: 

	sudo apt-get full-upgrade
Paket kurmak için: 

	sudo apt-get install paket-adı
Paket kaldırmak için: 

	sudo apt-get remove paket-adı
Depoda paket aramak için: 

	apt-cache search

Kullanıcı işlemleri

Yetkili kullanıcı olmak için: 

	sudo su

Dosya işlemleri

Dosya oluşturmak için: 

	touch dosya.uzantı
Klasör oluşturmak için: 

	mkdir klasör-adı
Dosya taşımak için: 

	mv ./dosya.uzantı /taşınacak/dizin/
Dosya kopyalamak için: 

	cp ./dosya.uzantı /kopyalanacak/dizin/
Dosya silmek için: 

	rm dosya.uzantı
